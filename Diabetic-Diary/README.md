# Diabetic Diary Console App

[English version below]

[PL]

Aplikacja ma na celu zbieranie i katalogowanie danych badań poziomu cukru we krwi pacjenta. 
Dane zapisywane są w pliku tekstowym. Aplikacja obsługiwana jest za pomocą terminala.

Projekt zrealizowany w ramach przedmiotu Informatyczne Systemy Medyczne (ISMED) 
w czasie studiów na kierunku Inżynieria Biomedyczna na Politechnice Warszawskiej.

**Autor**:
Emilia Poleszak (https://github.com/Emilia-Poleszak)

**Wymagania**:
Zainstalowany Python3.

Program nie korzysta z dodatkowych pakietów.

**Uruchomienie programu** 

Aby uruchomić program należy uruchomić plik main.py.



[ENG]

Diabetic Diary Console App collects and organises glucose test data and saves it in text file.
Project made during Biomedical Engineering studies, course Medical IT systems
("Informatyczne Systemy Medyczne" - ISMED) at Warsaw University of Technology.

**Author**:
Emilia Poleszak (https://github.com/Emilia-Poleszak)

**Requirements**:
Python3

This project does not use any package.

**How to start**

Run main.py
